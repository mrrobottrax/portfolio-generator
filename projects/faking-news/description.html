This game was made during a 48 hour game jam. Players connect using peer to peer to make up headlines and pick photos in this Jackbox inspired game.

(itch link provided, but by now I have no idea if the servers are still working)

  • Development time: 48 hours
  • Group size: 3
  • Date: 2023

Tools Used

Unity as the game engine, and Node JS to create the server used to connect players via lobbies.

  • Node JS

Development

Creating a peer to peer networking solution in 48 hours was a nightmare, and we had to cut a lot of the features we originally envisioned such as voice chat and streamed cameras (but it was close!).

A Node JS server keeps track of lobbies and generates room codes. After that, players are connected p2p.
