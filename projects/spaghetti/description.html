Development

Development is ongoing. So far it's been going well, but I'm sure we'll need to cut some stuff later on. This is by far the most ambitious game I've undertaken.

Features (as of now):

  • Custom netcode built using Steam Networking Sockets and TCP (for LAN mode)
  • Proximity chat
  • Easy game joining and integration with Steam
  • Procedurally generated mines
  • Custom render pipeline with stencil based realtime shadows
  • Deferred renderer optimized for our game
  • Fully synced inventory system

My responsibilities:

  • Netcode library
  • Steam integration
  • Procedural generation
  • Graphics programming
